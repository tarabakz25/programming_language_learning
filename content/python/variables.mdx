# 変数

## 概要

このセクションでは、Pythonにおける変数の使い方について学びます。変数はプログラミングの基礎となる重要な概念です。

## 解説

変数は、データを保存するための箱のようなものです。Pythonでは、変数に値を代入することで、その値を後で使用できます。変数名は自由に付けられますが、わかりやすい名前を付けることが推奨されます。

Pythonの変数には型の宣言が不要で、代入された値によって自動的に型が決まります。これを「動的型付け」と呼びます。

## 例

### 基本的な変数の使い方

\`\`\`python
# 文字列型の変数
name = "太郎"
print(name)  # 出力: 太郎

# 整数型の変数
age = 25
print(age)  # 出力: 25

# 浮動小数点型の変数
height = 175.5
print(height)  # 出力: 175.5

# 真偽値型の変数
is_student = True
print(is_student)  # 出力: True
\`\`\`

### 変数の再代入

\`\`\`python
x = 10
print(x)  # 出力: 10

x = 20
print(x)  # 出力: 20
\`\`\`

### 複数の変数への代入

\`\`\`python
# 複数の変数に同じ値を代入
a = b = c = 100
print(a, b, c)  # 出力: 100 100 100

# 複数の変数に異なる値を一度に代入
name, age, city = "花子", 30, "東京"
print(name, age, city)  # 出力: 花子 30 東京
\`\`\`
