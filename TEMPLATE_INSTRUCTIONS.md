# コンテンツ生成テンプレート

このドキュメントは、Cursorでプログラミング言語の学習コンテンツを生成する際のガイドラインです。

## 基本方針

1. **公式ドキュメントの構造を維持**: 対象言語の公式ドキュメントと同じセクション構成を保つ
2. **わかりやすい解説**: 専門用語を避け、初心者にも理解しやすい説明を心がける
3. **実践的な例**: 各概念に対して、実際に動作するコード例を必ず含める
4. **確認問題**: 各セクションの最後に3択または4択形式のクイズを追加

## MDXファイルの構造

```mdx
# セクションタイトル

## 概要

このセクションでは、[概念名]について学びます。

## 解説

[わかりやすく簡潔な説明]

## 例

\`\`\`[言語名]
[実際に動作するコード例]
\`\`\`
```

## セクション命名規則

- ファイル名は小文字とハイフンを使用（例: `variables.md`, `control-flow.md`）
- セクション名は公式ドキュメントの構造に合わせる

## 例: Python - Variables セクション

```mdx
# 変数

## 概要

このセクションでは、Pythonにおける変数の使い方について学びます。

## 解説

変数は、データを保存するための箱のようなものです。Pythonでは、変数に値を代入することで、その値を後で使用できます。

## 例

\`\`\`python
# 変数の定義
name = "太郎"
age = 25
height = 175.5

# 変数の使用
print(f"名前: {name}, 年齢: {age}, 身長: {height}cm")
\`\`\`
```

## 注意事項

- コード例は必ず動作するものを提供する
- 説明は簡潔に、重要なポイントを強調する
- クイズは理解度を確認できる適切な難易度にする
- 公式ドキュメントへのリンクを適宜追加する
